# Module heartbeat monitor with 6 inputs

> Detects and alerts if any of the monitored modules goes silent; meaning their heartbeat stops. Uses an AVR ATtiny2313 microcontroller.

Project documentation available at: https://cavelab.dev/wiki/Module_heartbeat_monitor_with_6_inputs

![Module heartbeat monitor with 6 inputs](https://cavelab.dev/images/0/08/Heartbeat-monitor-alarmed-4l00k4.jpeg)

Written in basic, using [BASCOM-AVR](http://www.mcselec.com/).
